Key Points:

  • Dynamic Creation: MCP endpoints created at startup from database configuration
  • Environment Control: Only agents listed in MCP_EXPOSED_AGENTS are exposed
  • Protocol Translation: MCP JSON-RPC → A2A Protocol → Agent Execution
  • Standard Interface: Any MCP-compatible client can access agents
  • Tool Format: Each agent exposes one tool: call_{agent_name}_agent
  • Endpoint Pattern: /agents/{agent_name}/mcp/* for all MCP operations
